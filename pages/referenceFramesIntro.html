Reference Frames Introduction

This portion of the website is meant to explain the basics of what a reference frame is, in 2-D space. The main points to explain are:
  • Coordinate systems and vectors for locations
  • Given's rotations
  • Object vs. Frame rotation
We'll tackle each of these ideas in the following sections:
  • Coordinate Systems, Reference Frames, and Vectors
  • Changing Coordinate Frames
  • Handling different coordinate frame axes
  • Combined rotation and translation
  • Frame vs Object Rotations
