About the Research

This website is meant to explain the basics of what a reference frame is in 2-D space, how to define the location of objects in that reference frame using vectors, and handling changes in coordinate frames. On this site you can also find representations of the following rotations in 3-D space:

  • Frame vs Object rotation
  • Cartesian coordinate system right-handed & left-handed rotations
  • Euler rotations
  • Full rotations in 3D
  • Yaw-Pitch-Roll rotation representations

Dr. Clark Taylor has provided research on these topics, and you'll find interactive examples within each paper to help you along the way.
